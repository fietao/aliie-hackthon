# GrocerySort - Quick Start Guide

Welcome to **GrocerySort** - your smart shopping list organizer!

## 🚀 Get Started in 60 Seconds

### Windows Users

1. **Run Setup** (first time only)
   ```
   Double-click setup.bat
   ```

2. **Start the App**
   ```
   Double-click run.bat
   ```

3. **Open in Browser**
   ```
   http://localhost:5000
   ```

### Mac/Linux Users

1. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

2. **Start the App**
   ```bash
   python app.py
   ```

3. **Open in Browser**
   ```
   http://localhost:5000
   ```

## 📝 How to Use

### Creating a List
1. Type an item (e.g., "milk", "chicken", "bread")
2. Press Enter or click "Add +"
3. Repeat for all items

### Organizing Your List
1. Click "CATEGORIZE" button
2. Items are automatically organized into categories
3. Each category has an icon for quick identification

### Starting Over
- Click "Home 🏠" on the results page to create a new list

## 🆘 Troubleshooting

### App Won't Start?
- Make sure Python is installed: `python --version`
- Install dependencies: `pip install -r requirements.txt`

### Wrong Category?
- Items are matched by keywords
- Try using the full product name
- Example: "ground beef" works better than "meat"

### Port Already in Use?
- Change the port in `run.bat` or command line:
  ```bash
  set PORT=8080 && python app.py
  ```
